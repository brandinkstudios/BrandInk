# BrandInk Studio

BrandInk Studio is a modern, dynamic web portfolio for a creative design agency. The website showcases the studio's projects, services, and core values through an engaging visual experience.

## Features

- **Modern UI/UX:** Built with sleek glassmorphism effects, dynamic hover states, and smooth CSS animations.
- **Responsive Design:** Fully mobile-friendly and optimized for different screen sizes.
- **Portfolio Gallery:** An interactive gallery section to filter and view past works (Websites, Posters, Logos) with a lightbox image viewer.
- **Service Listings:** Detailed breakdown of bespoke brand solutions and offerings.
- **Quick Contact:** Floating action buttons for immediate communication via WhatsApp, Instagram, or built-in chat UI.

## Technologies Used

- HTML5
- Vanilla CSS
- JavaScript
- Supabase (Backend/Database)

## Getting Started

To view the project locally, simply clone the repository and open `index.html` in your preferred web browser. No complex build tools or local servers are strictly required!

```bash
git clone https://github.com/priyaudayakumar8921/BrandInk.git
cd BrandInk
```
